从零开始搭建个人站

参考

  • 流程：https://zhuanlan.zhihu.com/p/60578464
  • 框架：https://hexo.io/zh-cn/
  • 主题：https://hexo.kira.host/

注意

主题相关的文件在：./node_modules/hexo-theme-kira

如果需要使用文章资源文件夹，需要在_config.yml中设置post_asset_folder: true

hexo常用命令

命令 功能
hexo new "name" # 新建文章
hexo new page "name" # 新建页面
hexo g # 生成页面
hexo d # 部署
hexo g -d # 生成页面并部署
hexo s # 本地预览
hexo clean # 清除缓存和已生成的静态文件
hexo help # 帮助

渲染latex公式

需要安装mathjax并换用pandoc渲染器

  • mathjax：mathjax
  • pandoc：pandoc

（注意，需要手动将pandoc添加进环境变量）

插入图片

参考：https://hexo.io/zh-cn/docs/asset-folders.html

注意：Front-matter部分的cover参数只能使用网络图片
